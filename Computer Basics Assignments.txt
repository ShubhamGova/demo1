6. which is the input devices that allows a user to move the cursor or pointer on the screen ?
Answer.
Mouse
A mouse is a hand-held pointing device that is widely used with computers and other digital devices. It typically consists of one or more buttons and a scrolling wheel.
In recent years, touchpads and trackballs have also become popular alternatives to traditional mice for cursor movement on laptops and some other devices. However, the traditional mouse remains a widely used and reliable input device for cursor control and interaction with computers.



7.which language is directly understood by the computer without a translation program?
Answer.
machine language or binary language
The language directly understood by a computer without a translation program is the machine language, also known as machine code. Machine language is a low-level programming language that consists of binary code, represented by 0s and 1s. Each instruction in machine language corresponds to a specific operation that the computer's central processing unit (CPU) can execute directly.
